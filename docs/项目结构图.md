## 一、整体项目结构

```
RuoYi-Vue/
├── 📁 bin/                          # 构建脚本目录
│   ├── clean.bat                    # 清理脚本
│   ├── package.bat                  # 打包脚本  
│   └── run.bat                      # 运行脚本
├── 📁 doc/                          # 文档目录
├── 📁 sql/                          # 数据库脚本
│   ├── ry_20250522.sql             # 初始化数据脚本
│   └── quartz.sql                   # 定时任务表脚本
├── 📄 pom.xml                       # Maven主配置文件
├── 📄 README.md                     # 项目说明文档
├── 📄 LICENSE                       # 开源协议
├── 🚀 ry.bat                       # Windows启动脚本
├── 🚀 ry.sh                        # Linux启动脚本
├── 📁 ruoyi-admin/                  # 🌟 启动模块 (Web入口)
├── 📁 ruoyi-common/                 # 🔧 通用工具模块
├── 📁 ruoyi-framework/              # 🏗️ 框架核心模块
├── 📁 ruoyi-system/                 # 👥 系统管理模块
├── 📁 ruoyi-quartz/                 # ⏰ 定时任务模块
├── 📁 ruoyi-generator/              # 🔨 代码生成模块
└── 📁 ruoyi-ui/                     # 🎨 前端Vue项目
```

## 四、核心文件说明

### 4.1 关键配置文件
| 文件路径 | 说明 | 作用 |
|---------|------|------|
| `pom.xml` | Maven主配置 | 定义项目依赖和构建配置 |
| `application.yml` | Spring Boot配置 | 应用程序主配置文件 |
| `application-druid.yml` | 数据源配置 | 数据库连接配置 |
| `package.json` | NPM配置 | 前端依赖和脚本配置 |
| `vue.config.js` | Vue CLI配置 | 前端构建配置 |

### 4.2 核心启动文件
| 文件路径 | 说明 | 作用 |
|---------|------|------|
| `RuoYiApplication.java` | 后端启动类 | Spring Boot应用入口 |
| `main.js` | 前端入口文件 | Vue应用入口 |
| `ry.bat` / `ry.sh` | 启动脚本 | 快速启动后端服务 |

### 4.3 重要业务文件
| 文件路径 | 说明 | 作用 |
|---------|------|------|
| `SecurityConfig.java` | 安全配置 | Spring Security配置 |
| `permission.js` | 权限控制 | 前端权限路由控制 |
| `router/index.js` | 路由配置 | 前端路由定义 |

## 五、数据流向图

```
🌐 浏览器
    ↓ HTTP请求
📱 Vue前端 (ruoyi-ui)
    ↓ API调用
🌐 Spring Boot后端 (ruoyi-admin)
    ↓ 业务处理
🏗️ 框架层 (ruoyi-framework)
    ↓ 权限控制
👥 业务层 (ruoyi-system)
    ↓ 数据访问
🗄️ 数据库 (MySQL)
    ↕️ 缓存
🔴 Redis缓存
```

## 六、模块依赖关系

```
ruoyi-admin (启动模块)
    ├── depends on → ruoyi-framework (框架模块)
    ├── depends on → ruoyi-system (系统模块)
    ├── depends on → ruoyi-quartz (定时任务模块)
    └── depends on → ruoyi-generator (代码生成模块)

ruoyi-framework (框架模块)
    └── depends on → ruoyi-common (通用模块)

ruoyi-system (系统模块)
    └── depends on → ruoyi-common (通用模块)

ruoyi-quartz (定时任务模块)
    └── depends on → ruoyi-common (通用模块)

ruoyi-generator (代码生成模块)
    └── depends on → ruoyi-common (通用模块)

ruoyi-common (通用模块)
    └── 基础模块，不依赖其他模块
```

## 七、技术栈对应关系

### 7.1 后端技术栈映射
```
📁 ruoyi-admin/          → Spring Boot (Web层)
📁 ruoyi-framework/      → Spring Security + Redis + 配置
📁 ruoyi-system/         → MyBatis + 业务逻辑
📁 ruoyi-common/         → 工具类 + 通用组件
📁 ruoyi-quartz/         → Quartz 定时任务
📁 ruoyi-generator/      → Velocity 代码生成
```

### 7.2 前端技术栈映射
```
📁 src/components/       → Element UI 组件封装
📁 src/views/           → Vue.js 页面组件
📁 src/router/          → Vue Router 路由
📁 src/store/           → Vuex 状态管理
📁 src/api/             → Axios HTTP请求
📁 src/utils/           → 工具函数
```
